Data cleanup

In this task, a manuipulated dataset was provided for cleanup process. I could identify several error types in the dataset using the Open Refine and Microsoft Excel. The error types included;

  • Empty records or rows in the middle of dataset.
  • Column adjustment or shifting
  • Total / Summation records in the middle of dataset
  • Spelling mistakes
  • Multiple representation
  • Duplicate records
  • Empty cell values
  • Wrong year, numeric values and ranges
  • Wrong year, numeric values and ranges

for more detailed about error types, errors and data cleaning process, click detailed anayalis.

Data visualization

D3 is a very famous Javascript library with visualization components to manipulate and bring data live. As a part of requirement for this task using d3, an interactive bar chart and grouped bar chart are used to represent data visualization. The bar chart depicts capacity in megawatt by energry (fuel) sources and the grouped bar chart allows you to compare two or more countries energy production sources and their capacity.
